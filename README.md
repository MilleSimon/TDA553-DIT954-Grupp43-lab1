# oopd-gu-chalmers Lab 2
This is the lab submission for the course Object-oriented Programming and Design, GU/Chalmers.
This submission includes a simple GUI for visualization of car positions and input control using Java swing.

## Description
The submission includes a wide variety of interactive vehicles including cars, trucks and car transporters. Factories are also present where vehicles can be stored.

### Controls
* Left and right arrow keys switch vehicles
* E for starting engine
* W for driving forward
* A for turning left
* D for turning right
* TODO add controls when we implement other vehicles
